# Insurance-Executive-Analytsis-Dashboard
Project Overview

The Insurance Analytics Dashboard is a Power BI project developed to analyze insurance business performance. It provides insights into premium revenue, claims, agent performance, risk exposure, and customer retention using interactive dashboards.

---

Problem Statement

Insurance companies generate large volumes of policy and claims data every day. Analyzing this data manually is time-consuming and can lead to delayed business decisions. This project aims to build an interactive Power BI dashboard that enables management to monitor key performance indicators (KPIs), identify business trends, and make data-driven decisions.

---

Objectives

- Monitor premium revenue and claim amounts.
- Track claim approval rates and settlement performance.
- Analyze agent performance.
- Identify high-risk policies and regions.
- Monitor customer retention and policy lapse rates.

---

Tools Used

- Power BI
- Power Query
- DAX
- Microsoft Excel

---

Dataset

The dataset contains insurance business information, including:

- Policy Details
- Customer Details
- Agent Details
- Claims Information
- Premium Amount
- Policy Status
- Claim Status
- Region
- Settlement Days

---

Data Cleaning

- Removed duplicate records.
- Handled missing values.
- Corrected data types.
- Renamed columns.
- Removed unnecessary columns.
- Standardized date formats.

---

Data Modeling

A Star Schema data model was created by connecting fact and dimension tables using appropriate relationships to improve reporting performance.
