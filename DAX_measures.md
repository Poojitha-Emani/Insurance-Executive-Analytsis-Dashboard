# 📐 DAX Measures - Insurance-Executive-Analytics-Dashboard

This document contains the key DAX measures used in the Insurance Analytics Dashboard.

---

## 💰 1. Total Premium

**Purpose:** Calculates the total premium collected from all policies.

```DAX
Total Premium =
SUM(Policies[Premium Amount])
```

---

## 📋 2. Total Claims Amount

**Purpose:** Calculates the total amount of claims.

```DAX
Total Claims Amount =
SUM(Claims[Claim Amount])
```

---

## 📄 3. Total Policies

**Purpose:** Counts the total number of insurance policies.

```DAX
Total Policies =
COUNT(Policies[Policy ID])
```

---

## ✅ 4. Active Policies

**Purpose:** Counts all active policies.

```DAX
Active Policies =
CALCULATE(
    COUNT(Policies[Policy ID]),
    Policies[Policy Status] = "Active"
)
```

---

## 📊 5. Total Claims Count

**Purpose:** Counts all insurance claims.

```DAX
Total Claims Count =
COUNT(Claims[Claim ID])
```

---

## ✔️ 6. Approved Claims Count

**Purpose:** Counts approved insurance claims.

```DAX
Approved Claims Count =
CALCULATE(
    COUNT(Claims[Claim ID]),
    Claims[Claim Status] = "Approved"
)
```

---

## ❌ 7. Rejected Claims Count

**Purpose:** Counts rejected insurance claims.

```DAX
Rejected Claims Count =
CALCULATE(
    COUNT(Claims[Claim ID]),
    Claims[Claim Status] = "Rejected"
)
```

---

## ⏳ 8. Pending Claims Count

**Purpose:** Counts pending insurance claims.

```DAX
Pending Claims Count =
CALCULATE(
    COUNT(Claims[Claim ID]),
    Claims[Claim Status] = "Pending"
)
```

---

## 📈 9. Claim Approval Rate %

**Purpose:** Calculates the percentage of approved claims.

```DAX
Claim Approval Rate % =
DIVIDE(
    [Approved Claims Count],
    [Total Claims Count],
    0
)
```

---

## 💵 10. Average Claim Amount

**Purpose:** Calculates the average claim amount.

```DAX
Average Claim Amount =
AVERAGE(Claims[Claim Amount])
```

---

## ⌛ 11. Average Settlement Days

**Purpose:** Calculates the average settlement time.

```DAX
Avg Settlement Days =
AVERAGE(Claims[Settlement Days])
```

---

## 📉 12. Loss Ratio %

**Purpose:** Measures the ratio of claims paid to premium collected.

```DAX
Loss Ratio % =
DIVIDE(
    [Total Claims Amount],
    [Total Premium],
    0
)
```

---

## 🔄 13. Claims Frequency

**Purpose:** Calculates the number of claims per policy.

```DAX
Claims Frequency =
DIVIDE(
    [Total Claims Count],
    [Total Policies],
    0
)
```

---

## 💸 14. Claims Severity

**Purpose:** Calculates the average cost per claim.

```DAX
Claims Severity =
DIVIDE(
    [Total Claims Amount],
    [Total Claims Count],
    0
)
```

---

## 👨‍💼 15. Premium per Agent

**Purpose:** Calculates the average premium generated by each agent.

```DAX
Premium per Agent =
DIVIDE(
    [Total Premium],
    DISTINCTCOUNT(Agents[Agent ID]),
    0
)
```

---

## 📑 16. Claims per Agent

**Purpose:** Calculates the average number of claims handled by each agent.

```DAX
Claims per Agent =
DIVIDE(
    [Total Claims Count],
    DISTINCTCOUNT(Agents[Agent ID]),
    0
)
```

---

## 📊 17. Agent Loss Ratio

**Purpose:** Calculates the loss ratio for each agent.

```DAX
Agent Loss Ratio =
DIVIDE(
    [Total Claims Amount],
    [Total Premium],
    0
)
```

---

## ⚠️ 18. Policy Lapse Rate %

**Purpose:** Calculates the percentage of lapsed policies.

```DAX
Policy Lapse Rate % =
DIVIDE(
    [Lapsed Policies],
    [Total Policies],
    0
)
```

---

## 📈 19. Premium Growth %

**Purpose:** Calculates premium growth compared with the previous year.

```DAX
Premium Growth % =
DIVIDE(
    [Total Premium] - [Premium LY],
    [Premium LY],
    0
)
```

---

# 🛠️ DAX Functions Used

- SUM()
- COUNT()
- CALCULATE()
- DIVIDE()
- AVERAGE()
- DISTINCTCOUNT()
- TOTALYTD()
- TOTALMTD()
- SAMEPERIODLASTYEAR()
